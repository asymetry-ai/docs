# API Reference

The Asymetry API provides endpoints for data ingestion, security analysis, and trace management. This reference documents the available endpoints and their usage.

## Base URL

All API endpoints are accessed via:

```
https://api.asymetry.co
```

---

## Authentication

All API requests require an API key passed in the `Authorization` header:

```bash
curl -X POST https://api.asymetry.co/v1/ingest \
  -H "Authorization: Bearer sk_prod_..." \
  -H "Content-Type: application/json" \
  -d '{"batch": [...]}'
```

### Getting an API Key

1. Sign in to your [Asymetry Dashboard](https://asymetry.co)
2. Navigate to **Settings** â†’ **API Keys**
3. Click **Create New Key**
4. Copy your key (it won't be shown again)

<Warning>
Keep your API key secret. Never commit it to version control or expose it in client-side code.
</Warning>

## Rate Limits

| Plan | Requests/minute | Traces/month |
| --- | --- | --- |
| Free | 60 | 10,000 |
| Pro | 600 | Unlimited |
| Enterprise | Custom | Custom |

Rate limit headers are included in responses:

```
X-RateLimit-Limit: 60
X-RateLimit-Remaining: 45
X-RateLimit-Reset: 1705312260
```

---

## Error Responses

API errors follow a consistent format:

```json
{
  "error": {
    "code": "invalid_api_key",
    "message": "The provided API key is invalid or expired.",
    "status": 401
  }
}
```

### Common Error Codes

| Code | Status | Description |
| --- | --- | --- |
| `invalid_api_key` | 401 | API key is missing or invalid |
| `rate_limit_exceeded` | 429 | Too many requests |
| `invalid_request` | 400 | Malformed request body |
| `internal_error` | 500 | Server error (retry with backoff) |

## Direct API Usage

If you need to call the API directly (e.g., from a non-Python environment):

<CodeGroup>
```bash curl
curl -X POST https://api.asymetry.co/v1/ingest \
  -H "Authorization: Bearer $ASYMETRY_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "batch": [{
      "type": "llm_request",
      "data": {
        "provider": "openai",
        "model": "gpt-4o",
        "messages": [{"role": "user", "content": "Hello"}],
        "response": [{"role": "assistant", "content": "Hi there!"}],
        "latency_ms": 1200
      }
    }]
  }'
```

```javascript JavaScript
const response = await fetch('https://api.asymetry.co/v1/ingest', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${process.env.ASYMETRY_API_KEY}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    batch: [{
      type: 'llm_request',
      data: { /* ... */ }
    }]
  })
});
```
</CodeGroup>
